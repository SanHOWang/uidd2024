# UIDD 2024 Project

## Getting started
1. 下載 [Bun](https://bun.sh/docs/installation)
2. 在終端機輸入
```bash
git clone https://github.com/yeeway0609/uidd2024.git
cd uidd2024
bun i
bun run dev
```
